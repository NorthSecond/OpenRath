# 示例

`example/` 目录提供可运行脚本。请在仓库根目录完成开发安装（`pip install -e .` 或 `uv sync`），并按[用户指南](../user_guide/index.md)顺序理解组件；每个示例在下方有**独立说明页**。

```{note}
会调用模型的示例依赖 `.env` 或已导出的 API 密钥；OpenSandbox 另需服务与配置。详见各页「依赖」一节。
```
